# Vector 1: Solo Operators & Their Stacks

**Question:** Who is actually running a solo or micro-team consulting/agency business powered by AI agents in 2025-2026? What services, tools, and scale?
**Agent dispatched:** 2026-03-12

## Findings

### Named Practitioners

1. **Joao Pedro Silva Setas** — 8 AI agent departments (CEO through Improver), GitHub Copilot custom agents + MCP memory server, 5 products on EUR 42/month infrastructure. Early stage (EUR 6.09 revenue after 2 months) but the most detailed public documentation of a solo agent-department architecture.

2. **Daniel Bilsborough** — Claude Code as core runtime, rejects all frameworks, uses markdown instruction files + Telegram bot + Mac Mini. Charges $5K+ per strategic assessment session.

3. **Nick Saraev** — AI automation agency educator (Make.com, N8N), targeting $2M/year profit, teaches others to build AI automation agencies.

4. **Jason Lemkin (SaaStr)** — Replaced 10 sales staff with 20 AI agents managed by 1.2 humans.

5. **Unnamed defense-tech founder** (Business Insider, Feb 2026) — 15 AI agent "council," saves 20 hrs/week.

### Revenue Benchmarks

- Solo AI content agency: $25K-$40K/month at 75-85% margins (8-12 clients)
- Solo retainer model: $9K-$12K/month (12 clients at $750-$1K each)
- AI advisory: $5K-$50K per engagement
- AI automation agency niches: $2K-$10K/month per client

### Strategic Insight

Every successful practitioner independently converged on markdown agent files + simple dispatch + human review — validating Sherpa's architecture. But nobody has structured governance (initiative lifecycle, behavioral constraints, quality gates, observable pipelines). That governance layer is Sherpa's genuine differentiation in this emerging market.

- The market is nascent — most practitioners are at EUR 6 to $40K/month scale
- Nobody has governance tooling; everyone is using ad-hoc markdown + manual review
- Sherpa's behavioral agent schema + initiative lifecycle fills an unoccupied gap
- The solo consulting model validates at $25K-$40K/month with 8-12 clients
