# website-content-v2 Research

## Summary

Research into authentic founder voice for sherpa.solar homepage rewrite. The v1 copy followed messaging guidelines correctly but reads as generic and interchangeable. Research focuses on how to find and express a distinctive voice grounded in real experience.

## Iterations

- **Iteration 1** (2026-03-21): Landscape survey — founder-voice sites, anti-template copy, practitioner authority patterns, voice discovery methods. Key finding: the voice already exists in the Foundation Stone; the website lost it by applying guidelines mechanically.

3. "I" vs "we" — which pronoun is right for a solo consultancy with the sherpa archetype?
4. Should governance data be surfaced live on the homepage rather than described?
