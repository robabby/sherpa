# Consulting Disruption Signals — Research

## Latest: Iteration 2 (2026-03-15)

Deep dive on three action-critical questions. Key finding: **Sherpa occupies an unclaimed layer between compliance (served by Big Four + platforms) and tooling (served by SDKs).** Convention-based governance — behavioral constraints, quality gates, lifecycle management — is the upstream factory that compliance consumes. Insurance is making governance a business requirement (hard requirement expected 2026-2027, following the cyber/MFA playbook). The engagement model is a taper: Build (3 months) → Operate (6 months) → Advise (9 months) → Sustain (ongoing).
