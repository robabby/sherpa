# sherpa-website Research

## Iteration 1 — Landscape Survey (2026-03-13)

Surveyed four domains: CMS/blog stack options, consulting website design patterns, dual-audience positioning, and Next.js technical state of art.

**Key conclusions:**
- **Stack:** Next.js 16 + Tailwind v4 + shadcn/ui + Velite (MDX day-one, Payload CMS Phase 2)
- **Positioning:** Anti-hype, "we built the tool we use," guide not guru
- **Architecture:** Shared hero with parallel CTAs, task-based navigation, content hubs over blog
- **Design:** Dark mode with warm accent, restrained animation, founder visibility
