ETH Zurich (February 2026, arXiv 2602.11988): LLM-generated context files reduce task success by ~3% and increase cost by 20%. Human-written files improve success by only ~4%. Context files should contain only truly non-inferable details. The generator script assembles human-written rules — it doesn't generate them.
