# Vector 4: AI/Tech-Specific Grants and Foundations

**Question:** What private foundations, tech accelerator grants, or AI-specific funding programs exist for early-stage technology companies in 2025-2026?
**Agent dispatched:** 2026-03-17

## Findings

### Open Source-Specific Grants (Highest Relevance)

- **NLnet Foundation (NGI Zero Commons Fund)** — Grants of EUR 5,000-50,000 (scalable higher) for open source projects that strengthen the open internet. New calls open every two months. **Current deadline: April 1, 2026.** Requirements: open licenses, open standards where possible, peaceful/humane purpose. Funded by European Commission's Next Generation Internet initiative. One of the strongest fits for Sherpa's open-source tooling.

- **Sovereign Tech Fund (Germany)** — Minimum EUR 50,000 per project for open digital base technologies: libraries, package managers, developer tools, protocol implementations. Rolling applications. Must be OSI/FSF-licensed. Does **not** fund prototypes or user-facing applications — focuses on infrastructure. Could apply if Sherpa frames studio-core/studio-mcp as developer infrastructure.

- **Open Technology Fund FOSS Sustainability Fund** — For established FOSS projects (3+ years old, regular updates, active user base). Recent grant: F-Droid received $396,044. Backed by Schmidt Futures' Plaintext Group, Omidyar Network, Okta, and GitHub. Sherpa would need to meet the maturity threshold.

- **GitHub Secure Open Source Fund** — $10,000 per project plus security training, Copilot access, and up to $150,000 in Azure credits via Microsoft for Startups. Backed by Sloan Foundation, Vercel, Stripe, Shopify. 2025 applications closed; watch for 2026 cycle.

### Cloud Credit Programs (Immediate, High Practical Value)

- **Google for Startups Cloud Program** — Up to $350,000 in Google Cloud credits for AI startups over 2 years ($250K year 1, $100K year 2). Includes $10K for partner LLM models. Self-serve tier: $2,000 for unfunded startups.

- **Microsoft for Startups Founders Hub** — Self-service $5,000 Azure credits (no VC required). With investor backing: up to $150,000. Includes $2,500 OpenAI credits, GitHub Enterprise, Visual Studio Enterprise licenses.

- **AWS Activate** — Up to $100,000 in credits for pre-Series B startups. AI-focused: up to $300,000. Generative AI Accelerator: up to $1M. Requires company website, founded within last 10 years, paid-tier AWS account.

- **Anthropic Startup Program** — $1,350 to $150,000+ in Claude API credits depending on tier. Top tiers do not require VC backing.

### AI-Specific Grant Programs

- **AI Grant (Nat Friedman / Daniel Gross)** — $250,000 funding + $250,000 Azure credits on founder-friendly SAFE terms (no cap, no discount). Rolling applications at aigrant.org. Note: technically a SAFE, so dilutive but maximally founder-friendly.

- **Anthropic Economic Futures Research Awards** — $10,000-$50,000 grants + $5,000 Claude API credits for empirical research on AI's economic impacts. Rolling applications. Six-month deliverable timeline. Sherpa's Human+AI workflow data could qualify.

- **Mozilla Foundation Incubator (Democracy x AI Cohort)** — $50,000 per project (10 projects), top 2 advance to $300,000 total. Open source component required. Deadline was March 16, 2026. Watch for future cohorts.

### Foundation Grants (Longer Shot, Larger Potential)

- **Alfred P. Sloan Foundation** — ~$80M/year. Technology program funds exploratory grantmaking in emerging tech, prototyping, proof-of-concept. Submit 2-page letter of inquiry. Has funded OSPOs (e.g., $650K to CMU).

- **Ford Foundation + Omidyar Network + Open Society Foundations** — Collaborative funding for critical digital infrastructure. $1.3M across 13 projects in recent round. Focus: diversity and sustainability in digital infrastructure.

- **Knight Foundation** — Primarily arts + tech and civic tech. AI and copyright research grants up to $20,000.

### General Small Business Grants

- **Arch Grants** — $75,000 non-dilutive + $25,000 relocation stipend (requires relocating to St. Louis). Applications through March 31, 2026.

- **Amex Shop Small / Verizon Digital Ready** — $5,000-$20,000. Low amounts but low effort.

### Programs Now Closed or Changed

- **Shuttleworth Foundation** — Shut down by early 2024.
- **Plaintext Group (Schmidt Futures)** — Appears archived.
- **CDAP (Canada)** — Wound down spring 2025.

## Implications

**Strongest immediate fits:**
1. **NLnet NGI Zero Commons Fund** — April 1, 2026 deadline. Sherpa's MCP server, studio-core, and behavioral agent framework are exactly the kind of open internet infrastructure NLnet funds. EUR 5K-50K with scaling potential. Should be the first application.
2. **Cloud credit programs** (Google, Microsoft, AWS, Anthropic) — Low-friction, immediate infrastructure value. Microsoft Founders Hub self-serve tier requires zero VC backing.
3. **Anthropic Economic Futures** — If Sherpa can produce empirical data on Human+AI collaborative productivity, rolling program offers $10-50K + API credits.

**Strategic framing matters:** Most open-source funds care about infrastructure and ecosystem impact, not consulting revenue. Applications should emphasize `@sherpa/studio` as reusable open-source infrastructure for Human+AI collaboration, not the consulting practice.

## Open Questions

- NLnet eligibility: does framework need to be published on npm before applying, or is "will be published" sufficient?
- AI Grant terms: $250K as SAFE — technically dilutive. Acceptable?
- Sovereign Tech Fund: user-facing app exclusion — does Studio app disqualify, or can application focus on core packages + MCP server?
- OTF maturity threshold: 3-year release history likely disqualifies Sherpa today
- Stacking: can NLnet + cloud credits + Anthropic research award be combined?
- Geographic restrictions: NLnet and Sovereign Tech Fund are EU-funded but accept international applicants — confirm US eligibility
