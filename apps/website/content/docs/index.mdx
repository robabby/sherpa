Sherpa is a collaboration framework for agentic workflows with a focus on governance, research, and execution — behavioral agent definitions, a dispatch pipeline, initiative lifecycle management, and executable conventions that live in your codebase and version with git.

## What you'll find here

- **Getting Started** — Install Sherpa, scaffold your project, and run your first dispatch
- **Concepts** — The mental models behind Sherpa: behavioral agents, governance, execution pipeline, conventions, and Studio
- **Guides** — Step-by-step walkthroughs for common tasks
- **Reference** — Configuration schema, CLI commands, MCP tools, component catalog
