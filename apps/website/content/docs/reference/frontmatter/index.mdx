---
title: Frontmatter Schemas
description: YAML frontmatter schemas for the governance artifacts Sherpa reads from your repository.
---

Every governance artifact is a markdown file with structured YAML frontmatter. These pages document each schema field by field.

- [Initiative](/docs/reference/frontmatter/initiative) — proposal status, lifecycle dates, risk, and relationships
- [Agent Role](/docs/reference/frontmatter/agent-role) — disposition, quality bar, and dispatch eligibility
- [Workstream](/docs/reference/frontmatter/workstream) — grouping for related initiatives
- [Rule](/docs/reference/frontmatter/rule) — glob patterns that control when conventions auto-load
