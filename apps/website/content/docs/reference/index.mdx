Exact, enumerable documentation for every public surface of the framework. The MCP tool and component pages are generated from source at build time, so they can't drift from the code.

- [CLI](/docs/reference/cli) — `sherpa init`, `sherpa sync`, and the full command surface
- [Configuration](/docs/reference/configuration) — the `sherpa.json` schema, section by section
- [Frontmatter](/docs/reference/frontmatter) — schemas for initiatives, agent roles, rules, and workstreams
- [MCP Tools](/docs/reference/mcp-tools) — every tool the Studio MCP server exposes, with parameters
- [Components](/docs/reference/components) — the `@sherpa/studio-ui` component catalog, grouped by domain
