## 🆕 Product Launches & Updates

### Google Antigravity (Live)
- Launched Nov 2025 alongside Gemini 3; now fully documented and compared in the wild
- Pricing clarified: Free tier, $20/mo Pro (bundled with Google AI Pro), $249.99/mo Ultra
- Available in VS Code, JetBrains, and 35+ platforms
- **Key differentiator:** Fully free base tier; multi-model access (Claude, Codex, Gemini)

### Amazon Kiro (Active)
- AWS's agentic IDE; emphasizes **spec-driven development** — structured requirements docs before AI writes code
- Features: "specs," "steering files," "hooks" (event-triggered agent actions)
- Auto agent mode launched late 2025
- Had a security bulletin (March 17) — arbitrary code execution via crafted project files; patched
- **Key differentiator:** Spec-first workflow enforces discipline before AI acts; AWS ecosystem integration

### Windsurf (Cognition)
- Wave 13 update: Arena Mode (blind model comparison), Plan Mode, parallel multi-agent sessions
- Quota-based pricing now documented
- Proprietary SWE-1.5 model reportedly 13× faster than Claude Sonnet 4.5
- March 2026 satisfaction rankings: **Claude Code leads satisfaction**, Cursor leads ARR

### The New Postman
- Relaunched as "AI-native, built for the agentic era"
- Git-native workspaces, API Catalog, Agent Mode, multi-protocol support
- Relevant: APIs are increasingly consumed by agents, not humans — Postman pivoting to match
