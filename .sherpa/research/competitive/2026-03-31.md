## 🤖 AI-Native IDEs & Coding Agents

### Cursor — $2B ARR, $30B Valuation, Uncertain Future
- **Fortune deep-dive (Mar 21):** Cursor hit **$2B in annual recurring revenue** and a **$30B valuation**, but the piece raises real questions about the company's moat as Anthropic (their model provider and now a competitor via Claude Code) increasingly competes directly.
- **Current pricing:** $29/month Pro. Dominates on raw capability — best-in-class VS Code + JetBrains support, real-time collaboration, Supermaven autocomplete.
- **Sherpa angle:** Cursor is the dominant *individual developer* tool. Sherpa Studio's play is *team/workflow orchestration*, which Cursor doesn't address.

### Google Antigravity — Full-Stack Vibe Coding Launch (Mar 18–30)
- **Announced Nov 2025**, now shipping fast: Mar 18 launch of **full-stack vibe coding** in Google AI Studio, integrated with Antigravity agent + Firebase backend.
- Natural language → production-ready full-stack apps. One-click pathway from AI Studio → Antigravity.
- Google also shipped **"Agent Smith"** internally (leaked), described as "a junior developer on call 24/7" — reportedly went viral inside Google after early 2026 launch.
- **Benchmark:** Antigravity hitting **76.2% SWE-bench** (per morphllm comparison).
- **Sherpa angle:** Google is attacking the *creation* layer (build apps from prompts). Sherpa Studio targets the *engineering process* layer — more complementary than directly competitive today, but watch the trajectory.

### AWS Kiro — Spec-Driven IDE, Now in SageMaker
- **Kiro** (Amazon's AI IDE, powered by Claude Sonnet) launched its **Students program** (free 1-year, 1K credits/month) on Mar 18.
- **Mar 26:** AWS SageMaker Studio now supports Kiro and Cursor as remote IDEs — deepens the AWS lock-in story.
- Core differentiator: **spec-first development** — AI writes requirements doc before code. Free tier at 50 interactions/month; Pro at $19/month.
- **Security note:** A CVE for arbitrary code execution via crafted project files was patched Mar 17.
- **Sherpa angle:** Kiro's spec-first workflow is the closest direct conceptual overlap with Sherpa Studio's structured engineering approach. Worth monitoring closely.

### Windsurf — JetBrains Support Lands, Still Price Competitive
- March 2026: **JetBrains support** ships, removing the VS Code lock-in objection.
- Pricing: $49/month Pro (vs Cursor $29/month) — a gap that's hard to justify unless EU compliance or Flow-state awareness matters to the buyer.
- SWE-1.5 model described as fast; "80% of Cursor's capability at 75% of the price" per buildmvpfast review.

### Claude Code — $2.5B ARR, 4% of GitHub Commits
- **Anthropic total ARR:** ~$19B by March 2026 (up from $14B in February). Claude Code alone: **$2.5B annualized run rate** as of Feb 2026, more than doubling since Jan 1.
- **~300,000 business customers** on Claude Code.
- **~4% of all GitHub public commits** now authored by Claude Code.
- The framing: *"With Claude Code you operate, with Devin you delegate."* Claude Code = interactive/terminal-native; Devin = async/delegated.
- **Sherpa angle:** Claude Code is the underlying engine Rob is building on/with. Its explosive growth validates the market. Sherpa Studio as an *orchestration layer* on top of Claude Code is a strong position.

### Devin (Cognition) — Most Autonomous, 67% PR Merge Rate
- Still the benchmark for **fully autonomous coding**: own sandboxed IDE, browser, terminal, shell.
- **67% PR merge rate** on defined tasks.
- Competitive framing has settled: Devin = delegation, Claude Code = operation. Both winning different use cases.

## 🎯 Implications for Sherpa Studio

1. **Kiro is the closest conceptual competitor** — spec-first, structured engineering workflow. Amazon distribution is a moat but also a ceiling. Sherpa Studio can be more opinionated and faster-moving.

2. **Accenture's FDE practice is the enterprise threat** — big consultancy with Microsoft co-sell. Counter: Sherpa's speed, founder-led expertise, and agentic-native approach vs. retrofitted traditional consulting.

3. **The Claude Code wave is a tailwind** — $2.5B ARR and 300K business customers validate the market Rob is building for. Sherpa Studio as an orchestration/workflow layer on top of this is well-positioned.

4. **"Vibe coding" is a positioning risk** — the term is going mass-market (non-technical users). Sherpa Studio needs clear language that differentiates *agentic engineering workflow* from "prompt-to-app" tools like Replit and Lovable. The audience is engineers and engineering teams, not non-technical builders.

5. **Google Antigravity trajectory to watch** — full-stack vibe coding + Firebase + Workspace integration is a powerful surface. If they add team workflow features, this becomes more competitive at the product layer.
