# Competitive Landscape: AI Engineering & Agentic Dev Tools
**Week of March 28, 2026**

## 🔥 Top Signal This Week

The agentic IDE market exploded this week. JetBrains launched *two* products (Air + Central), Cursor shipped Composer 2, Anthropic gave Claude Code computer-use abilities, and Reflection AI is eyeing a $25B valuation. The market is consolidating around "agentic platform" as the dominant positioning — exactly where Sherpa Studio plays.

## 🧰 Agentic IDE & Coding Tools

### Cursor — Automations + Composer 2
- **Automations** launched March 5: always-on agents that run dev tasks on autopilot with human-in-the-loop checkpoints. Predecessor "Bugbot" was the prototype.
- **Composer 2** launched March 19: frontier-class agentic coding model, positioned as both powerful *and* budget-friendly.
- 📌 *Sherpa angle:* Cursor is doubling down on autonomous background agents — the same territory Sherpa Studio occupies. Their moat is IDE lock-in; ours is workflow intelligence and consulting depth.

### JetBrains — Air (Public Preview) + Central (Announced)
- **Air** (based on abandoned Fleet codebase) went to public preview March 10 — macOS only, Windows/Linux coming. Supports Claude Agent, OpenAI Codex, Gemini CLI, and JetBrains Junie. Concurrent multi-agent task execution.
- **Central** announced March 24 — agentic platform with governance, cloud infra for running agents, shared cross-repo context, token management (JetBrains Console), and usage analytics. Retiring "Code With Me" (pair programming) in favor of agent orchestration. EAP in Q2 2026 with design partners.
- 📌 *Sherpa angle:* JetBrains Central is governance + observability for agent fleets — that's close to what Sherpa Studio's orchestration layer does. They're targeting enterprise IntelliJ shops. Opportunity to position Sherpa as the "Air/Central for teams not locked into JetBrains."

### Windsurf (Codeium)
- SWE-1.5 model is now on their paid plans — optimized for rapid agent code gen.
- Cascade remains their flagship: predictive context + autonomous multi-file editing, claims 13x faster than Claude 3.5 Sonnet for certain workflows.
- Pricing updated with credit-based system (1 credit = ~4 GPT-4.1 prompts).
- 📌 *Sherpa angle:* Windsurf is beginner-friendly and IDE-first. Not a direct threat to Sherpa Studio's workflow orchestration play, but competes for developer mindshare.

### Claude Code (Anthropic)
- **Auto mode** launched ~March 23: expanded autonomy with guardrails.
- **Computer use**: Claude Code + Cowork can now open files, use the browser, and run dev tools (March 23).
- **Code Review**: automatic pre-commit bug catching.
- **Dispatch for Cowork**: delegate tasks to agents async.
- Version 2.1.76 (March 14) brought rapid-fire improvements.
- 📌 *Sherpa angle:* Anthropic is building a full async developer workflow suite. Claude Code's trajectory is toward replacing junior devs on ticket work — positions premium consulting (Sherpa) as the differentiator for strategy, architecture, and transformation.

### Xcode 26.3 (Apple)
- Native agentic coding support powered by Anthropic Claude Agent and OpenAI Codex.
- Directly targets iOS/macOS devs — native first-party agentic IDE is now table stakes on Apple platforms.
- 📌 *Sherpa angle:* Interesting for WavePoint (6 Apple apps), but not a Sherpa Studio threat per se.

### Gemini Code Assist + CLI (Google)
- Gemini CLI provides terminal-native agent workflows.
- Gemini Code Assist integrates into IDE environments.
- Growing fast — positioned as the enterprise Google Workspace play.
