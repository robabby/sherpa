## 🤖 AI Coding Tools

### Cursor — 🔥 Breaking: Cursor 3 Launches
**Released April 2, 2026**

- **Cursor 3** (codename: Glass) launched with a new **agent-first interface**
- Natural language prompt box where developers describe tasks → AI agents complete them autonomously
- Agents spin up in **cloud sandboxes** with access to MCP configs and user-selected models
- Agents have a **memory tool** — they learn from past runs and improve with repetition
- Framing: direct response to Claude Code and OpenAI Codex

**Architecture shift:** Cursor's third era — devs describe intent, agents plan + implement + test + demo end-to-end. Human = reviewer/decision-maker.

### Claude Code (Anthropic)
Latest updates (April 2, 2026):

- **`/powerup` interactive lessons** — in-product teaching moments
- **MCP tool result persistence override** via `_meta["anthropic/maxResultSizeChars"]` annotation (up to 500K chars) — lets large DB schemas pass without truncation
- `disableSkillShellExecution` setting added
- `--bare` flag for scripted `-p` calls (skips hooks, LSP, plugin sync, skill walks; requires API key)
- `--channels` flag added
- **Retiring**: 1M token context window beta for Claude Sonnet 4.5 and Claude Sonnet 4 on April 30, 2026 — if you're relying on that, plan accordingly
