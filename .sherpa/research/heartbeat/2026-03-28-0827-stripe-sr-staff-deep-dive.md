## Interview Process (Senior Staff Specifics)

- **Recruiter screen** → **Technical phone screen** (coding, practical not LeetCode)
- **On-site (virtual), ~5-6 rounds:**
  - Coding (2 rounds) — practical, build-it style problems
  - System design (1-2 rounds) — frontend system design: dashboards, component systems, real-time data architectures
  - Debugging round — find and fix bugs in existing code
  - **Presentation round** ⚡ (Staff+ only) — Write ~1000-word one-pager about a past project, present 20 min to staff-level engineer + note-taker, deep Q&A follows
  - Behavioral/hiring manager — leadership, influence, cross-team impact
- **Key difference at Senior Staff:** Deeper architectural scrutiny, more emphasis on cross-org influence, presentation round carries more weight. They want org-wide technical decisions, not just team-level.
