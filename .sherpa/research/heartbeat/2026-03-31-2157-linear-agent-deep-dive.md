## What Linear Just Shipped (March 24, 2026)

### The Manifesto: "Issue Tracking Is Dead"
- CEO Karri Saarinen published a letter declaring the end of issue tracking as a category
- Core thesis: Traditional issue tracking was built for a **handoff model** — PMs scope work, engineers pick it up later, and ceremony fills the gap. AI changes that dynamic fundamentally.
- Linear is repositioning from "issue tracker" to **"product development system for teams and agents"**
- In 2024: zero issues created by agents. By early 2026: **one in four issues** created by agents. Trajectory is clear.

### Linear Agent (Beta)
A native agent interface that works across online, mobile, desktop, and as a plugin:
- **Product context awareness** — works within your workspace's full context
- **User feedback analysis** — can analyze and synthesize customer feedback
- **Project/issue/document creation** — crafts structured project artifacts from natural language
- **Code-aware tasks** — can diagnose app functionality, design technical specifications
- **Non-technical accessibility** — "teammates can ask questions they'd normally have to track down an engineer to answer"

### Skills System
- Reusable, codifiable workflows — "as you find workflows worth repeating, you can codify them as reusable skills"
- Triggered via **slash commands** or **automatically** when Linear determines context warrants it
- Compound learning — teams build institutional knowledge into the agent
- **Same concept as Figma Skills** (launched the day before, March 23) — "Skills" is becoming the standard agent interface pattern in 2026

### Linear Coding Agent (Upcoming)
- Linear will write code and automatically fix bugs
- Powered by frontier models + native Linear context and tools
- **Code Diffs** — code review directly inside Linear, interface for both humans and agents
- Builds on earlier work: Triage Intelligence + deep integrations with cloud coding agents

### Automations
- Agent workflows trigger the moment issues enter triage
- Every new issue adds context; Linear intelligently refines, synthesizes, or takes action on arrival
- Automatic deduplication, classification, scope suggestion, task sequencing
- Can draft specs, generate checklists, update project status
