# Research

Overnight research collected by Luna via scheduled cron jobs.

## Directories

- `job-market/` — Frontend/engineering job postings mentioning AI/agentic, emerging titles, skills in demand
- `competitive/` — Who's publicly talking about AI-accelerated dev workflows, what they're calling themselves, thought leadership

## Format

Each research run produces a dated markdown file: `YYYY-MM-DD.md`

Files include: summary, raw findings, links visited, sources, and a "so what" synthesis for Rob's positioning.

## Purpose

Feed Rob's job search strategy and personal branding. Eventually surface in Sherpa Studio.
